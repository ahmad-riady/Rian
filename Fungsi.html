Fungsi

Fungsi Python adalah blok pernyataan yang melakukan tugas tertentu. Idenya adalah menggabungkan beberapa tugas yang umum atau berulang kali dilakukan dan membuat sebuah fungsi sehingga alih-alih menulis kode yang sama berulang kali untuk input yang berbeda, kita dapat melakukan pemanggilan fungsi untuk menggunakan kembali kode yang ada di dalamnya berulang kali.

Jika suatu code memerlukan suatu sub code yang sama, maka cara yang paling mudah adalah mengcopy-paste sub code tersebut. Akan tetapi cara ini akan menyulitkan programmer untuk debugging, maintenance suatu code. Oleh karena itu jika suatu code akan semakin kompleks, diperlukan fitur tambahan untuk code yang semakin kompleks ini dengan tujuan :

  • mempermudah programmer ketika proses debugging code
  • mempermudah programmer untuk maintenace code (edit, update code)
  • memperkecil ukuran code
  • mempermudah programmer untuk menggunakan kembali sub code yang sama pada code

Fitur tambahan ini dinamakan dengan Fungsi/Function/Procedure. Fungsi adalah suatu fitur yang dimiliki oleh bahasa pemrograman yang membungkus suatu sub code dengan nama fungsi atau prosedur tertentu.

Di dalam python, untuk membuat fungsi, diperlukan keywords def.

Untuk memanggil fungsi pada python, perlu didefinisikan terlebih dahulu sebelumnya.

Beberapa hal yang harus diperhatikan pada fungsi ini:

  • argument/parameter : nilai yang dikirim oleh pemanggil fungsi
  • return value : nilai balik yang dihasilkan oleh fungsi
  • local atau Global Variable

Argumen

Informasi dapat diteruskan ke fungsi sebagai argumen.

Argumen ditentukan setelah nama fungsi, di dalam tanda kurung. Anda dapat menambahkan argumen sebanyak yang Anda inginkan, cukup pisahkan dengan koma.

Kita akan membuat fungsi sederhana dalam Python untuk memeriksa apakah angka yang dimasukkan sebagai argumen ke fungsi tersebut genap atau ganjil.

In[4]:
def evenOdd(x):
    if (x % 2 == 0):
        return "Even"
    else:
        return "Odd"

print(evenOdd(16))
print(evenOdd(7))

Even
Odd

Jumlah Argumen

Secara default, suatu fungsi harus dipanggil dengan jumlah argumen yang benar.

Jika fungsi Anda mengharapkan 2 argumen, Anda harus memanggilnya dengan tepat 2 argumen.

Parameter vs Argumen

Istilah parameter dan argumen dapat digunakan untuk hal yang sama: informasi yang diteruskan ke suatu fungsi.

Jenis Argumen Fungsi

Python mendukung berbagai jenis argumen yang dapat diteruskan saat pemanggilan fungsi. Dalam Python, terdapat beberapa tipe argumen fungsi berikut. Mari kita bahas satu per satu.

1. Argumen Default

Argumen default adalah parameter yang mengasumsikan nilai default jika nilai tidak diberikan dalam pemanggilan fungsi untuk argumen tersebut.

In[22]:
def my_function(name = "friend"):
  print("Hello", name)

my_function("Emil")
my_function("Tobias")
my_function()
my_function("Linus")

Hello Tobias
Hello friend
Hello Linus

3. Argumen Posisional

Dalam argumen posisi, nilai ditetapkan ke parameter berdasarkan urutannya dalam pemanggilan fungsi.

4. Argumen Sembarangan

Dalam Argumen Kata Kunci Sembarangan Python, *args dan **kwargs dapat meneruskan sejumlah variabel argumen ke suatu fungsi menggunakan simbol khusus. Ada dua simbol khusus:

  • *args dalam Python (Argumen Non-Kata Kunci)
  • **kwargs dalam Python (Argumen Kata Kunci)

Kode ini secara terpisah menampilkan argumen non-kata kunci (*args) dan kata kunci (**kwargs) dalam fungsi yang sama.

In[31]:
def myFun(*args, **kwargs):
    print("Non-Keyword Arguments (*args):")
    for arg in args:
        print(arg)

    print("\nKeyword Arguments (**kwargs):")
    for key, value in kwargs.items():
        print(f"{key} == {value}")

# Function call with both types of arguments
myFun('Hey', 'Welcome', first='Geeks', mid='for', last='Geeks')

Non-Keyword Arguments (*args):
Hey
Welcome

Keyword Arguments (**kwargs):
first == Geeks
mid == for
last == Geeks

*args

Jika Anda tidak tahu berapa banyak argumen yang akan dimasukkan ke dalam fungsi Anda, tambahkan a * sebelum nama parameter.

Dengan cara ini, fungsi akan menerima sejumlah argumen dan dapat mengakses item yang sesuai

Apa itu *args?
Parameter ini *args memungkinkan suatu fungsi menerima sejumlah argumen posisi.

Di dalam fungsi, args menjadi tupel yang berisi semua argumen yang dilewatkan

Contoh
Mengakses argumen individual dari *args:

**kwargs

Jika Anda tidak tahu berapa banyak argumen kata kunci yang akan dimasukkan ke dalam fungsi Anda, tambahkan dua tanda bintang ** sebelum nama parameter.

Dengan cara ini, fungsi akan menerima kamus argumen dan dapat mengakses item yang sesuai

Apa itu **kwargs?
Parameter ini **kwargsmemperbolehkan suatu fungsi menerima sejumlah argumen kata kunci.

Di dalam fungsi tersebut, kwargsmenjadi kamus yang berisi semua argumen kata kunci:

Local Variable

Local variabel adalah variabel yang dikenali didalam fungsi itu sendiri saja

Variabel yang dibuat di dalam suatu fungsi termasuk dalam cakupan lokal fungsi tersebut, dan hanya dapat digunakan di dalam fungsi tersebut.

In[37]:
def setPassword():
    password='tanggalLahirku'
    return password
def displayPassword(password):
    print('the password is',password)
#Main
passWd=setPassword()
displayPassword(passWd)

Global Variable

Suatu variabel yang dapat dikenali oleh semua bagian dari code termasuk semua fungsi

Variabel yang dibuat di bagian utama kode Python adalah variabel global dan termasuk dalam cakupan global.

Variabel global tersedia dari dalam cakupan apa pun, global dan lokal.

Kata Kunci Global

Jika Anda perlu membuat variabel global, tetapi terjebak dalam lingkup lokal, Anda dapat menggunakan "global" kata kunci.

Kata "global" kunci membuat variabel menjadi global.

In[38]:
total=0
def jumlah(a,b):
    c=a+b
    global total
    total=total+c
jumlah(6,7)
print(total)

13

Kata Kunci Nonlokal

Kata nonlocal kunci digunakan untuk bekerja dengan variabel di dalam fungsi bersarang.

Kata nonlocal kunci membuat variabel menjadi milik fungsi luar.

In[39]:
def myfunc1():
  x = "Jane"
  def myfunc2():
    nonlocal x
    x = "hello"
  myfunc2()
  return x

print(myfunc1())

hello

Dekorator Python

Dekorator memungkinkan Anda menambahkan perilaku tambahan ke suatu fungsi, tanpa mengubah kode fungsi.

Dekorator adalah fungsi yang mengambil fungsi lain sebagai input dan mengembalikan fungsi baru.

In[40]:
def changecase(func):
  def myinner():
    return func().upper()
  return myinner

@changecase
def myfunction():
  return "Hello Sally"

print(myfunction())

HELLO SALLY

Python Lambda

Fungsi Lambda
Fungsi lambda adalah fungsi anonim kecil.

Fungsi lambda dapat mengambil sejumlah argumen, tetapi hanya dapat memiliki satu ekspresi.

Sintaksis
lambda arguments : expression
Ekspresi dieksekusi dan hasilnya dikembalikan:

In[41]:
def myfunc(n):
  return lambda a : a * n

mydoubler = myfunc(2)

print(mydoubler(11))
