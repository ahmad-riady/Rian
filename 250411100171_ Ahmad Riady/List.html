Dalam Python, list adalah struktur data bawaan yang dapat menampung kumpulan item yang berurutan. Tidak seperti array dalam beberapa bahasa pemrograman, list dalam Python sangat fleksibel:

  • Dapat berisi item duplikat
  • Dapat diubah: item dapat dimodifikasi, diganti, atau dihapus
  • Diurutkan: mempertahankan urutan penambahan item
  • Dapat menyimpan tipe data campuran (bilangan bulat, string, boolean, bahkan daftar lainnya)
  • Masing-masing elemen dipisahkan oleh ','.
  • Didalam list terdapat index yang menunjukkan elemen ke-berapa dari suatu list.
  • Index dari list ini dimulai dari '0'.
  • elemen-elemenpada list ini dibuka dan ditutup dengan kurung siku '[ ]', [value, value, value, ...]

Membuat List

List dapat dibuat dengan beberapa cara, seperti menggunakan tanda kurung siku, konstruktor list(), atau dengan mengulang elemen. Mari kita bahas setiap metode satu per satu dengan contoh:

1. Menggunakan Tanda Kurung Siku

Kita menggunakan tanda kurung siku [] untuk membuat daftar secara langsung.

In[3]:
a = [1, 2, 3, 4, 5]
b = ['apple', 'banana', 'cherry'] 
c = [1, 'hello', 3.14, True]
print(a)
print(b)
print(c)

[1, 2, 3, 4, 5]
['apple', 'banana', 'cherry']
[1, 'hello', 3.14, True]
2. Menggunakan Konstruktor list()

Kita juga dapat membuat daftar dengan meneruskan suatu iterable (seperti tuple , string atau daftar lainnya) ke fungsi list() .

In[4]:
a = list((1, 2, 3, 'apple', 4.5))  
print(a)

b = list("GFG")
print(b)

[1, 2, 3, 'apple', 4.5]
['G', 'F', 'G']
3. Membuat Daftar dengan Elemen Berulang

Kita dapat menggunakan operator perkalian * untuk membuat daftar dengan item yang berulang.

In[5]:
a = [2] * 5
b = [0] * 7

print(a)
print(b)

[2, 2, 2, 2, 2]
[0, 0, 0, 0, 0, 0, 0]
Seperti halnya pada string, index digunakan untuk mengakses suatu elemen pada list. Index dimulai dari '0' untuk mengakses elemen pertama pada list.
  • List [start:end]
  • List[:end]
  • List[start:]
  • List[:]
  • len(list)
In[9]:
nama=['ria','anton','dina','dita','amir','rina','doni']

print(nama[4:7])
print(nama[:3])
print(nama[6:])
print(nama[:])

['amir', 'rina', 'doni']
['ria', 'anton', 'dina']
['doni']
['ria', 'anton', 'dina', 'dita', 'amir', 'rina', 'doni']
Untuk menentukan berapa banyak item yang dimiliki suatu daftar, gunakan len()
In[10]:
thislist = ["apple", "banana", "cherry"]
print(len(thislist))

Menambahkan Elemen ke dalam List

Kita dapat menambahkan elemen ke dalam List menggunakan metode berikut:

  • append() : Menambahkan elemen di akhir List.
  • extend() : Menambahkan beberapa elemen ke akhir List.
  • insert() : Menambahkan elemen pada posisi tertentu.
In[12]:
a = []

a.append(10)  
print("After append(10):", a)  

a.insert(0, 5)
print("After insert(0, 5):", a) 

a.extend([15, 20, 25])  
print("After extend([15, 20, 25]):", a)

After append(10): [10]
After insert(0, 5): [5, 10]
After extend([15, 20, 25]): [5, 10, 15, 20, 25]

Menghapus Elemen dari List

Kita dapat menghapus elemen dari List menggunakan:

  • remove(): Menghapus kemunculan pertama suatu elemen.
  • pop(): Menghapus elemen pada indeks tertentu atau elemen terakhir jika tidak ada indeks yang ditentukan.
  • pernyataan del: Menghapus elemen pada indeks tertentu.
In[1]:
a = [10, 20, 30, 40, 50]

a.remove(30)  
print("After remove(30):", a)

popped_val = a.pop(1)  
print("Popped element:", popped_val)
print("After pop(1):", a) 

del a[0]  
print("After del a[0]:", a)

After remove(30): [10, 20, 40, 50]
Popped element: 20
After pop(1): [10, 40, 50]
After del a[0]: [40, 50]

Mengulang list

Anda dapat melakukan pengulangan pada item-item dalam list dengan menggunakan for loop.
Anda juga dapat melakukan pengulangan pada item-item list dengan mengacu pada nomor indeksnya.

Gunakan fungsi range() dan len() untuk membuat iterasi yang sesuai.

In[2]:
thislist = ["apple", "banana", "cherry"]
for i in range(len(thislist)):
  print(thislist[i])

apple
banana
cherry

Menggunakan Perulangan While
Anda dapat melakukan pengulangan pada item-item dalam list dengan menggunakan whileloop.

Gunakan len() fungsi tersebut untuk menentukan panjang list, lalu mulai dari 0 dan lakukan pengulangan melalui item list dengan merujuk pada indeksnya.

Ingatlah untuk menambah indeks sebesar 1 setelah setiap iterasi.

In[3]:
thislist = ["apple", "banana", "cherry"]
i = 0
while i < len(thislist):
  print(thislist[i])
  i = i + 1

apple
banana
cherry

Urutkan List

Objek list memiliki sort() metode yang akan mengurutkan list secara alfanumerik, menaik, secara default:

In[4]:
thislist = ["orange", "mango", "kiwi", "pineapple", "banana"]
thislist.sort()
print(thislist)

['banana', 'kiwi', 'mango', 'orange', 'pineapple']
  • Urutkan Menurun
    Untuk mengurutkan secara menurun, gunakan argumen kata kunci reverse = True:
In[5]:
thislist = ["orange", "mango", "kiwi", "pineapple", "banana"]
thislist.sort(reverse = True)
print(thislist)

['pineapple', 'orange', 'mango', 'kiwi', 'banana']
  • Urutan Terbalik Bagaimana jika Anda ingin membalik urutan list, apa pun alfabetnya?
    Metode ini reverse() membalik susunan penyortiran elemen saat ini.
In[6]:
thislist = ["banana", "Orange", "Kiwi", "cherry"]
thislist.reverse()
print(thislist)

['cherry', 'Kiwi', 'Orange', 'banana']

Salin Daftar

Anda tidak dapat menyalin daftar hanya dengan mengetik list2 = list1, karena: list2hanya akan menjadi referensi ke list1, dan perubahan yang dibuat di list1akan secara otomatis juga dibuat di list2.

Gunakan metode copy() Anda dapat menggunakan metode Daftar bawaan copy()untuk menyalin daftar.

In[7]:
thislist = ["apple", "banana", "cherry"]
mylist = thislist.copy()
print(mylist)

['apple', 'banana', 'cherry']

Cara lain untuk membuat salinan adalah dengan menggunakan metode bawaan list().

In[8]:
thislist = ["apple", "banana", "cherry"]
mylist = list(thislist)
print(mylist)

['apple', 'banana', 'cherry']

Gabung Daftar

Ada beberapa cara untuk menggabungkan, atau menyatukan, dua atau lebih daftar dalam Python.

Salah satu cara termudah adalah dengan menggunakan + operator.

In[9]:
list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

list3 = list1 + list2
print(list3)

['a', 'b', 'c', 1, 2, 3]

Cara lain untuk menggabungkan dua daftar adalah dengan menambahkan semua item dari daftar2 ke daftar1, satu per satu:

In[10]:
list1 = ["a", "b" , "c"]
list2 = [1, 2, 3]

for x in list2:
  list1.append(x)

print(list1)

['a', 'b', 'c', 1, 2, 3]

Atau Anda dapat menggunakan extend() metode ini, yang tujuannya adalah untuk menambahkan elemen dari satu daftar ke daftar lainnya:

In[11]:
list1 = ["a", "b" , "c"]
list2 = [1, 2, 3]

list1.extend(list2)
print(list1)

['a', 'b', 'c', 1, 2, 3]
