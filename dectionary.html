Dictionary

Python Dictionary

Dictionary adalah struktur data yang menyimpan nilai dalam pasangan kunci:nilai. Nilai dalam dictionary dapat berupa tipe data apa pun dan dapat diduplikasi, sedangkan kunci tidak dapat diulang dan harus tidak dapat diubah.

  • Kunci bersifat peka huruf besar-kecil, yang berarti nama yang sama tetapi huruf besar-kecil pada Kunci akan diperlakukan secara berbeda.
  • Kunci harus tidak dapat diubah, artinya kunci dapat berupa string, angka atau tupel, tetapi bukan list.
  • Kunci duplikat tidak diizinkan dan kunci duplikat apa pun akan menimpa nilai sebelumnya.
  • Secara internal menggunakan hashing. Oleh karena itu, operasi seperti pencarian, penyisipan, dan penghapusan dapat dilakukan dalam Waktu Konstan.

Copy Dictionaries

Salin dictionary

Anda tidak dapat menyalin dictionary hanya dengan mengetik dict2 = dict1, karena: dict2 hanya akan menjadi referensi ke dict1, dan perubahan yang dibuat dalam dict1akan secara otomatis juga dibuat dalam dict2.
Ada beberapa cara untuk membuat salinan, salah satunya adalah dengan menggunakan metode dictionary bawaan copy().

conntoh:

In[12]:
thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
mydict = thisdict.copy()
print(mydict)

{'brand': 'Ford', 'model': 'Mustang', 'year': 1964}

Cara lain untuk membuat salinan adalah dengan menggunakan fungsi bawaan dict().

In[13]:
thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
mydict = dict(thisdict)
print(mydict)

{'brand': 'Ford', 'model': 'Mustang', 'year': 1964}

Dictionary Methods

Method dictionary

  • clear() = Menghapus semua elemen dari dictionary
  • copy() = Mengembalikan salinan dari dictionary
  • fromkeys() = Mengembalikan dictionary baru dengan key tertentu dan satu nilai default
  • get() = Mengembalikan nilai dari key yang ditentukan tanpa error jika key tidak ada
  • items() = Mengembalikan pasangan keyvalue dalam bentuk tuple
  • keys() = Mengembalikan semua key dalam dictionary
  • pop() = Menghapus elemen berdasarkan key dan mengembalikan nilainya
  • popitem() = Menghapus dan mengembalikan pasangan keyvalue terakhir
  • setdefault() = Mengembalikan nilai key; jika belum ada, menambahkan key dengan nilai default
  • update() = Memperbarui dictionary dengan pasangan keyvalue dari sumber lain
  • values() = Mengembalikan semua nilai dalam dictionary
