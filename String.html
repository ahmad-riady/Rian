String

Dalam Python, string adalah serangkaian karakter yang diapit tanda kutip. String dapat berisi huruf, angka, simbol, atau spasi. Karena Python tidak memiliki tipe karakter terpisah, bahkan satu karakter pun dianggap sebagai string dengan panjang satu. String banyak digunakan untuk penanganan dan manipulasi teks.
Membuat String String dapat dibuat menggunakan tanda kutip tunggal ('...') atau ganda ("...") . Keduanya berfungsi sama.

string merupakan salah satu tipe data dalam suatu bahasa pemrograman yang terdiri dari beberapa character.

Periksa String

Untuk memeriksa apakah frasa atau karakter tertentu ada dalam suatu string, kita dapat menggunakan kata kunci in.

In[7]:
txt = "The best things in life are free!"
print("free" in txt)

True

Periksa jika TIDAK

Untuk memeriksa apakah frasa atau karakter tertentu TIDAK ada dalam suatu string, kita dapat menggunakan kata kunci not in.

In[8]:
txt = "The best things in life are free!"
print("expensive" not in txt)

String Slicing

Slicing adalah cara untuk mengekstrak sebagian string dengan menentukan indeks awal dan akhir . Sintaks untuk pengirisan adalah string[mulai:akhir] , di mana mulai adalah indeks awal dan akhir adalah indeks akhir (dikecualikan).
Contoh: Dalam contoh ini kita mengiris rentang dan membalikkan string.

In[4]:
s = "GeeksforGeeks"
print(s[1:4])    # characters from index 1 to 3
print(s[:3])     # from start to index 2
print(s[3:])     # from index 3 to end
print(s[::-1])   # reverse string

Immutable String

String bersifat immutable, artinya string tidak dapat diubah setelah dibuat. Jika kita perlu memanipulasi string, kita dapat menggunakan metode seperti penggabungan, pemotongan, atau pemformatan untuk membuat string baru berdasarkan string asli.

Contoh: Dalam contoh ini kita mengubah karakter pertama dengan membuat string baru.

In[5]:
s = "geeksforGeeks"
s = "G" + s[1:]   # create new string
print(s)

GeeksforGeeks

Penggabungan String

Untuk menggabungkan, atau mengkombinasikan, dua string, Anda dapat menggunakan operator +.

In[1]:
a = "Hello"
b = "World"
c = a + b
print(c)

HelloWorld

Menghapus String

Dalam Python, menghapus karakter individual dari sebuah string tidak mungkin dilakukan karena string bersifat immutable. Namun, kita dapat menghapus seluruh variabel string menggunakan kata kunci del .

Contoh: Di sini, kita menggunakan kata kunci del untuk menghapus string.

In[10]:
s = "GfG"
del s

Memperbarui String

Karena string tidak dapat diubah, pembaruan membuat string baru menggunakan pemotongan atau metode seperti replace() .

Contoh: Kode ini memperbaiki huruf pertama dan mengganti kata.

In[11]:
s = "hello geeks"
s1 = "H" + s[1:]                   # update first character
s2 = s.replace("geeks", "GeeksforGeeks")  # replace word
print(s1)
print(s2)

Hello geeks
hello GeeksforGeeks

Format String

Seperti yang kita pelajari di bab Variabel Python, kita tidak dapat menggabungkan string dan angka Tetapi kita dapat menggabungkan string dan angka dengan menggunakan f-string atau format() metode!

In[3]:
age = 36
txt = f"My name is John, I am {age}"

Karakter Escape

Untuk menyisipkan karakter yang ilegal dalam suatu string, gunakan karakter escape.

Karakter escape adalah garis miring terbalik \yang diikuti oleh karakter yang ingin Anda masukkan.

Contoh karakter ilegal adalah tanda kutip ganda di dalam string yang diapit tanda kutip ganda
Untuk memperbaiki masalah ini, gunakan karakter escape ":

In[5]:
txt = "We are the so-called \"Vikings\" from the north."
print(txt)
