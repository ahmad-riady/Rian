Tuple

Tuple digunakan untuk menyimpan beberapa item dalam satu variabel.

Tuple adalah salah satu dari 4 tipe data bawaan dalam Python yang digunakan untuk menyimpan kumpulan data, 3 lainnya adalah List , Set , dan Dictionary , semuanya dengan kualitas dan penggunaan yang berbeda.

Tuple dalam Python adalah kumpulan elemen yang berurutan dan tidak dapat diubah.

Tuple mirip dengan list, tetapi tidak seperti list, tuple tidak dapat diubah setelah dibuat (yakni, tidak dapat diubah).
Tuple dapat menampung elemen dengan tipe data yang berbeda.

Item Tuple

Item tupel bersifat teratur, tidak dapat diubah, dan memperbolehkan nilai duplikat.
Item tuple diindeks, item pertama memiliki indeks [0], item kedua memiliki indeks, [1]dan seterusnya.

Dipanggil

Ketika kita katakan bahwa tupel diurutkan, artinya item-itemnya memiliki urutan yang pasti, dan urutan itu tidak akan berubah.

Tidak dapat diubah

Tuple tidak dapat diubah, artinya kita tidak dapat mengubah, menambah atau menghapus item setelah tuple dibuat.

Membuat Tuple

Tuple dibuat dengan menempatkan semua item di dalam tanda kurung (), dipisahkan dengan koma. Tuple dapat memiliki sejumlah item dan dapat memiliki tipe data yang berbeda .

# Using List
li = [1, 2, 4, 5, 6]
print(tuple(li))

# Using Built-in Function
tup = tuple('Geeks')

Memperbarui Tuple

Tuple tidak dapat diubah, artinya Anda tidak dapat mengubah, menambah, atau menghapus item setelah tuple dibuat.

Namun ada beberapa solusi.

Ubah Nilai Tuple Setelah sebuah tupel dibuat, Anda tidak dapat mengubah nilainya. Tupel tidak dapat diubah , atau disebut juga immutable .

Namun, ada solusinya. Anda dapat mengonversi tupel menjadi daftar, mengubah daftar, dan mengonversi daftar kembali menjadi tupel.

  • Tambahkan Item

    Karena tupel tidak dapat diubah, tupel tidak memiliki metode bawaan append(), tetapi ada cara lain untuk menambahkan item ke tupel.
  1. Ubah menjadi list : Sama seperti solusi untuk mengubah tupel, Anda dapat mengubahnya menjadi list, menambahkan item Anda, dan mengubahnya kembali menjadi tupel.
In[8]:
thistuple = ("apple", "banana", "cherry")
y = list(thistuple)
y.append("orange")
thistuple = tuple(y)
print(thistuple)

('apple', 'banana', 'cherry', 'orange')
  1. Tambahkan tuple ke tuple . Anda diperbolehkan menambahkan tuple ke tuple. Jadi, jika Anda ingin menambahkan satu item (atau banyak), buatlah tuple baru dengan item tersebut, lalu tambahkan ke tuple yang sudah ada:
In[9]:
thistuple = ("apple", "banana", "cherry")
y = ("orange",)
thistuple += y

print(thistuple)

('apple', 'banana', 'cherry', 'orange')

  • Hapus Item

    Catatan: Anda tidak dapat menghapus item dalam tupel.

Tuple tidak dapat diubah , jadi Anda tidak dapat menghapus item darinya, tetapi Anda dapat menggunakan solusi yang sama seperti yang kami gunakan untuk mengubah dan menambahkan item tuple:

In[11]:
thistuple = ("apple", "banana", "cherry")
y = list(thistuple)
y.remove("apple")
thistuple = tuple(y)
print(thistuple)

('banana', 'cherry')

Atau Anda dapat menghapus tupel sepenuhnya:

Penggabungan Tuple

Tuple dapat digabungkan menggunakan operator +. Operasi ini menggabungkan dua atau lebih tuple untuk menghasilkan tuple baru.

In[14]:
tup1 = (0, 1, 2, 3)
tup2 = ('Geeks', 'For', 'Geeks')

tup3 = tup1 + tup2
print(tup3)

(0, 1, 2, 3, 'Geeks', 'For', 'Geeks')

Dalam Python, operator "" dapat digunakan dalam pembongkaran tupel untuk mengambil beberapa item ke dalam sebuah daftar. Ini berguna ketika Anda ingin mengekstrak hanya beberapa elemen tertentu dan menggabungkan sisanya.

tup = (1, 2, 3, 4, 5)

a, *b, c = tup

print(a)

Tuple Perulangan

Anda dapat melakukan pengulangan melalui item tupel dengan menggunakan for perulangan.
Anda juga dapat melakukan pengulangan melalui item tupel dengan merujuk pada nomor indeksnya.

Gunakan fungsi range()dan len()untuk membuat iterasi yang sesuai.

In[17]:
thistuple = ("apple", "banana", "cherry")
for i in range(len(thistuple)):
  print(thistuple[i])

apple
banana
cherry

Menggunakan Perulangan While
Anda dapat melakukan pengulangan melalui item tupel dengan menggunakan whileperulangan.

Gunakan len()fungsi tersebut untuk menentukan panjang tupel, lalu mulai dari 0 dan lakukan pengulangan melalui item tupel dengan merujuk pada indeksnya.

Ingatlah untuk menambah indeks sebesar 1 setelah setiap iterasi.

In[18]:
thistuple = ("apple", "banana", "cherry")
i = 0
while i < len(thistuple):
  print(thistuple[i])
  i = i + 1

apple
banana
cherry
